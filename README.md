
# ♟️ Juego de Ajedrez en Java

Aplicación de escritorio desarrollada en Java utilizando Swing.
Implementa diseño orientado a objetos, gestión del estado del tablero e interacción basada en eventos.
---

## Descarga y ejecución

Descarga el archivo ejecutable desde la sección ejecutable:

Ajedrez(Swing).jar

Cómo ejecutarlo:

Asegúrate de tener Java instalado (JDK 8 o superior).

    java -jar ChessGame.jar

Si tu sistema lo permite, también puedes ejecutarlo directamente haciendo doble clic sobre el archivo .jar.
